This branch contains **Java GUI layout examples** demonstrating different ways to arrange components in a graphical user interface.  
It is designed for learning and reference purposes for BSCS students.

---

## üìÇ Folder Structure


| Layout      | Description |
|------------|-------------|
| No Layout  | Demonstrates absolute positioning of components. Good for understanding manual component placement. |
| Flow       | FlowLayout examples. Arranges components in a row, flowing to the next line if needed. Useful for toolbars, buttons, or simple panels. |
| Grid       | GridLayout examples. Arranges components in rows and columns. Useful for forms, calculators, or evenly spaced buttons. |
| Border     | BorderLayout examples. Placement in North, South, East, West, Center. Ideal for main windows with panels, menus, and headers. |
| Card       | CardLayout examples. Allows switching between multiple panels in the same space. Useful for wizards, multi-step forms, or tab-like navigation. |
